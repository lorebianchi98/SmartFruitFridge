# SmartFruitFridge
This project is developed for the university course of "Internet of Things".
Smart Fruit Fridge is an automation application for the smart management of a fruit fridge. It could be useful in a logistics company or for greengrocers to manage the conservation and ripening of fruit, it controls an MQTT network that was deployed at the Department of Information Engineering at Universit√† di Pisa with Zolertia RE-Mote sensor nodes. It also controls a CoAP network emulated using Cooja Emulator.
